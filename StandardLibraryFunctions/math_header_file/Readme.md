# The `math.h` header file

- `ceil`: Ceiling
- `floor`: Floor
- `fabs`: Absolute value of floating point
- `log`: Base `e` logarithm
- `pow`: Power
- `sqrt`: Square root
- `exp`: Exponential. Returns the value of `e` raised to the `x`th power.
- `isnan`: Is not a number
- `isfinite`: Is a finite number
